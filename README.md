# Capture-Seq Analysis for Thesis Chapter 2

This repository contains the analysis scripts for **Chapter 2 of my PhD thesis**.

The analyses presented here were used to benchmark isoform detection methods and investigate **differential isoform regulation in the human brain** using targeted Oxford Nanopore cDNA Capture-Seq.

## ⚙️ Requirements

- **R ≥ 4.2** with:
  - `IsoformSwitchAnalyzeR`
  - `tidyverse`
  - `data.table`
  - `ggpmisc`
- **Python ≥ 3.8** with:
  - `pysam`
- **External tools**:
  - [Minimap2](https://github.com/lh3/minimap2)
  - [Samtools](http://www.htslib.org/)
  - [TAMA](https://github.com/GenomeRIK/tama)
  - [SQANTI3](https://github.com/ConesaLab/SQANTI3)
